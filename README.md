# keiba-physics-ocp
JRA全場対応｜**解析力学×最適制御×調教スペクトル（FFT/DFT）**で「所要時間最小化」を解く競馬予想モデル。

## 目的
- コースの勾配・曲率・幅・直線/カーブ構成を場別プロファイル化
- 抵抗（転がり/空気/勾配）＋摩擦円（コーナーでの縦横合成限界）を入れた**時間最小化OCP**を解く
- 調教1年の**FFT/Lomb–Scargle**（週周期・位相・HF/LF・エントロピー）＋4点DFT（ラップ形状）で **CP/W'/Pmax の事前**を与える
- 各馬パラメータ推定→当該レース予測→ボルツマン・ソフトマックスで勝率化→印/買い目へ
